## Описание проекта

Веб-приложение «Калькулятор», разработанное на чистом **JavaScript (Vanilla JS)** без использования сторонних фреймворков и библиотек.

### Основные функции

- Арифметические операции: сложение, вычитание, умножение, деление
- Операция процента (`%`)
- Цепочка операций (например `5 + 3 * 2` — при вводе второго оператора сначала считается предыдущее)
- Удаление последнего символа (кнопка `⌫`)
- **Валидация**: деление на ноль показывает сообщение об ошибке, ввод ограничен 12 символами
- **История вычислений** — хранится как массив объектов, отображается от новых к старым
- **Фильтрация истории** по типу операции (`+`, `−`, `×`, `÷`)
- Поддержка ввода с клавиатуры
- Адаптивный интерфейс (работает на мобильных)

## Установка и запуск

> ⚠️ Проект использует ES-модули (`import/export`), поэтому файл `index.html` **нельзя открыть напрямую в браузере** (будет ошибка CORS). Нужен локальный сервер.

### Вариант 1 — Live Server (VS Code)

1. Установите расширение **Live Server** в VS Code
2. Откройте папку `calculator` в VS Code
3. Правая кнопка на `index.html` → **Open with Live Server**
4. Браузер откроется автоматически

### Вариант 2 — через Node.js

```bash
npx serve .
```

Затем откройте адрес из терминала (обычно `http://localhost:3000`).

## Примеры использования

### Базовое вычисление

```
Нажмите: 1 2 + 8 =
Результат: 20
```

### Деление на ноль

```
Нажмите: 5 ÷ 0 =
Результат: сообщение "Деление на ноль невозможно", дисплей сбрасывается
```

### Процент

```
Нажмите: 2 0 0 %
Результат: 2  (200 / 100)
```

### Фильтрация истории

Выберите оператор в выпадающем списке справа — в истории останутся только вычисления с этим оператором.

### Клавиатура

| Клавиша | Действие |
|---------|----------|
| `0`–`9` | ввод цифр |
| `+` `-` `*` `/` | операторы |
| `Enter` или `=` | вычислить |
| `Backspace` | удалить последний символ |
| `Escape` | сброс (`C`) |
| `.` | десятичная точка |
| `%` | процент |

## Покрытие требований проекта

| Требование | Реализация |
|-----------|-----------|
| Работа с DOM | Динамическое обновление дисплея, генерация списка истории через `createElement` |
| Обработчики событий | Клики (делегирование), ввод с клавиатуры (`keydown`), `change` на select |
| Валидация + сообщения об ошибках | Деление на ноль, некорректный ввод — `showError()` |
| Массив объектов | `History.items` — массив объектов `{ id, expression, result, operator, time }` |
| Фильтрация | `History.filter(op)` — фильтрует массив по оператору |
| Модульность | Три модуля: `calculator.js`, `history.js`, `ui.js` |
| ES6+ | Классы, модули, стрелки, spread, template literals, деструктуризация |
| Адаптивность | CSS Flexbox + `@media (max-width: 640px)` |
| Интерактивность | Поддержка клавиатуры, история с фильтром |
